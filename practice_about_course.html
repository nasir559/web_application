Welcome to the Web Application Course

About Course

Well, we can guess the topics and learnings which will be covered in this course by seeing the name of the course, which is WEB APPLICATION. We will learn how to make a website using Html, CSS, JavaScript. We will also make our website publically visible using Netlify so that others can also see our website. Moreover, we will also learn about Github.

Core Course Topics

HTML

Building the structure of the web pages.

CSS

Styling, colors, Flexbox, and Grid layouts.

JavaScript

Adding interactivity and logic to the application.
